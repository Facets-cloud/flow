# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc] as its
code of conduct. The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

In short: be kind, be patient, assume good faith, and focus on the work.
Criticism is welcome when it's about the code or the design; personal
attacks are not. We want flow to be a place where new contributors feel
safe asking questions and disagreeing in public.

## Reporting

If you see behaviour that violates the Code of Conduct — in an issue,
PR, comment, or any other project space — open a GitHub issue and tag
the maintainers. We'll look at it.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
